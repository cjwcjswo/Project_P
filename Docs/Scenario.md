# 📜 [마나 위버스] 메인 시나리오 기획서 (v1.0)

## 1. 로그라인 (Logline)
> "마왕의 깽판으로 마법이 사라져 '물리력'만 남은 대환장 세계! 대정령(유저)이 되어 퍼즐을 맞춰, 든든한 열혈 바보 기사 '테오'와 오합지졸 동료들에게 마나를 떠먹여 세상을 구원하는 유쾌발랄 퍼즐 RPG."

## 2. 세계관 핵심 설정 (Lore)
* **마나 침묵 (Mana Silence):** 마왕 보이드가 세계의 동력원인 '마나 코어'를 부숴버려서 세상의 모든 마법과 스킬이 봉인된 상태. 속성의 개념조차 사라져버렸다.
* **영웅들의 처지:** 대마법사도, 성기사도 스킬을 못 써서 몬스터를 상대로 눈물겨운 '육탄전(기본 공격)'을 벌이고 있다.
* **대정령 (유저):** 부서진 마나 코어의 파편(퍼즐 블록)을 맞춰, 억지로 마나 에너지를 만들어내는 전무후무한 능력자. 대정령이 퍼즐을 맞춰줘야만 영웅들이 기적처럼 스킬을 쓸 수 있다.

## 3. 프롤로그: 마나 침묵과 대정령의 기상

**[배경 연출]**
화면 전체가 흑백으로 변하며 웅장하지만 어딘가 나사가 빠진 BGM이 흐른다. 거대한 마왕 '보이드'가 세계의 중심에 있던 반짝이는 구슬(마나 코어)을 망치로 냅다 깨부수는 컷신.

**[내레이션]**
*"그날, 세계의 마나가 멈췄다."*
*"마왕 보이드의 수작으로 대륙에는 '마나 침묵'이 찾아왔다."*
*"불을 뿜던 대마법사의 지팡이는 그저 값비싼 막대기가 되었고, 힐러들은 성수 대신 빨간약을 발라야 했다."*
*"영웅들은 이제... 몬스터를 상대로 눈물겨운 '육탄전'을 벌여야만 했다."*

**[화면 전환]**
아주 조그마한 슬라임을 상대로 무거운 대검을 낑낑대며 휘두르는 전사 **'테오'**의 안습한 모습. 데미지는 고작 **[-1], [-1]** 씩 뜨고 있다.

**[테오의 말풍선]**
*"헉... 헉... 검이 왜 이렇게 무겁지...? 스킬... 스킬 마렵다...!"*

**[내레이션]**

*"일어나라, 위대한 **'대정령(Player)'**이여! 저 답답하고 불쌍한 바보들에게 퍼즐로 마나를 떠먹여 세상을 구원하라!"*

## 4. 메인 캐릭터 소개

### 🌟🌟🌟 왕국 제일의 열혈 바보, 테오 (초기 지급 3성 전사)
* **포지션:** Index 0 (최전방 / 🔴 빨간색 블록)
* **설명:** 유저와 끝까지 함께할 메인 파트너. 과거엔 왕국 최강의 기사였지만, 지금은 그저 체력 좋고 긍정적인 '듬직한 바보 형'이다. 대정령(유저)을 절대적으로 믿고 따르며, 퍼즐이 터질 때마다 가장 격하게 환호한다.
* **특이사항:** 유일하게 궁극기(컷인 연출)를 보유한 채로 시작하여 튜토리얼을 가이드함.

### 🌟 약초 빻는 사제, 토비 (챕터 1 합류 / 1성 힐러)
* **포지션:** Index 1 (2열 / 🟡 노란색 블록)
* **설명:** 신성력이 끊긴 후, 진짜 약초를 빻아 만든 즙을 아군에게 뿌리며 버티고 있는 생계형 사제. 겁이 많고 소심하지만, 노란색 블록이 매칭되는 순간만큼은 전설의 성자 빙의해서 붕대를 감아준다.

### 🌟 물리력 만렙 마법사, 유리 (챕터 2 합류 / 1성 마법사)
* **포지션:** Index 2 (3열 / 🟢 초록색 블록)
* **설명:** "마법이 안 나가면 지팡이로 때리면 되지!"를 몸소 실천 중인 성격 까칠한 마법사. 묵직한 마도서 모서리로 몬스터의 뚝배기를 깨는 능력이 일품이다. 초록색 블록이 맞춰지면 툴툴거리면서도 화려한 광역 마법을 쏟아붓는다.

## 5. 시나리오 로드맵

* **Chapter 1: 슬라임 동산의 기적**
    * 테오와 대정령의 운명적 만남. 3성 테오의 압도적 위력을 체험. 최소 인원을 채우기 위해 근처에서 약 팔던 '토비'를 섭외함.
* **Chapter 2: 오크 숲의 깡패들**
    * 지팡이로 오크와 영혼의 맞다이를 뜨던 '유리' 구출. 드디어 3인 기본 파티(빨, 노, 초) 완성. 유저는 1성 동료들의 궁극기 부재를 느끼며 소환의 필요성을 깨닫게 됨.
* **Chapter 3: 차원의 문이 열리다**
    * 대정령의 힘으로 '소환 상점' 개방. 4~5인 덱을 구성하며 본격적으로 마왕군의 핵심 간부들을 하나씩 참교육하기 시작함.
* **에필로그 향:** * 오합지졸이었던 테오, 토비, 유리가 대정령의 '퍼즐 먹방' 케어를 통해 전설의 영웅들로 거듭나며 마침내 마왕 보이드의 뚝배기를 깨러 성으로 진격함.
