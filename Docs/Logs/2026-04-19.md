## IDE 진단 참고 사항

`GameManager.cs`의 HeroDataRepository/SkillDataRepository "not found" 오류는
파일 생성 직후 IDE 캐시 미갱신으로 인한 **false positive**.
`.asmdef` 파일 없음 확인 → 전체 스크립트 동일 어셈블리 소속 → Unity Reimport 후 자동 해결.
